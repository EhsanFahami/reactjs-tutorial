## Instruction

1. install `react` and `react-dom`

```
yarn add react react-dom --save
```

2. install `babel-preset-react`

```
yarn add babel-preset-react --dev
```

3. add `react` as presets in .babelrc file

4. change HelloWorld component

```
const HelloWorld = () => (
  <div>Hello World</div>
);
```
